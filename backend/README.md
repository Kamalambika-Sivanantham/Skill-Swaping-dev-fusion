# Skill-Swaping-dev-fusion

## Overview

SkillSwap is a peer-to-peer skill exchange platform where users teach skills they know and learn skills they want without monetary transactions.

## Problem Statement

Many students and learners cannot afford expensive courses or lack access to mentors. SkillSwap enables users to exchange knowledge directly through skill-sharing sessions.

## Core Features

* User Profiles
* Skill Matching Engine
* Real-time Chat
* Session Scheduling
* Video Session Support
* Ratings & Reviews
* Badge System
* Community Leaderboard
